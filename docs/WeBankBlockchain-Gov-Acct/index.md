# 账户治理组件
[![License](https://img.shields.io/badge/license-Apache%202-4EB1BA.svg)](https://www.apache.org/licenses/LICENSE-2.0.html)

```eval_rst
.. admonition:: **简介**

    WeBankBlockchain-Gov-Acct定位为区块链账户治理中间件，旨在充分合理地在区块链节点网络中，利用智能合约所提供的图灵完备的计算能力，提供自洽的账户治理能力
```

```eval_rst
.. admonition:: **主要特性**

    - 支持多种治理方式
    - 去中心化的分布式协作治理思想
    - 账户全生命周期的治理
    - 全面、灵活的集成方式
    - 支持可插拔的设计，对业务侵入小
    - 支持国密
    - 支持社交好友重置私钥
```
